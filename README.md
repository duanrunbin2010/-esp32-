# 基于esp32的智慧种植装置
##### 注：本装置由我本人（一位准初二的学生）耗时一个月独自制作的，因此可能会有很多不足之处，望谅解°(°¯᷄◠¯᷅°)°｡

### 本项目可实现以下功能
#### 1）通过微信小程序远程查看土壤温度，土壤湿度，光照强度，环境温湿度，太阳能电压等功能
#### 2）通过微信小程序远程开启或关闭灌溉系统
#### 3）通过太阳能电池板等实现自理，无需额外充电
#### 4）通过OLED显示屏实时查看当前环境温湿度，土壤湿度
#### 5）可通过手机配网

### 本项目所需主要硬件
#### 1）ESP32-wroom 
#### 2）电容式土壤温湿度传感器
#### 3）LM35温度传感器
#### 4）DHT11温湿度传感器
#### 5）0.96寸OLED显示屏（IIC接口）
#### 6）3.7V软包锂电池
#### 7）太阳能充放电模块，太阳能电池板
#### 8）光敏电阻
